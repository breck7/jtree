Release Notes

Here's a list of the notable changes in JTree.

61.2.0 2023-1-5

  • 🎉 Performance improvements to core (typed map method).

61.1.0 2023-1-5

  • 🎉 Performance improvements to core.

61.0.1 2023-1-4

  • 🏥 TreeBase: SearchServer now exported correctly

61.0.0 2023-1-4

  • ⚠️ Grammar: BREAKING: contentDelimiter is now listDelimiter
  • 🎉 Grammar: New keyword uniqueLine to check for duplicate lines.

60.0.0 2022-12-12

This is an unstable release. Please wait a few days for bug fix releases before using.

  • ⚠️ BREAKING: TreeBaseServer is now exported from treeBaseServer.node.js and has been rewritten
  • ⚠️ BREAKING: SearchServer is now exported from treeBaseServer.node.js
  • 🎉 New class: TreeBaseBuilder and rewritten class TreeBaseServer

59.1.2 2022-12-10

  • 🏥 TreeBase: copy fix in search engine

59.1.1 2022-12-03

  • 🏥 TreeBase: bug fixes in rank and webPermalink

59.1.0 2022-12-02

  • 🎉 TreeBase: SearchServer now shipped in JTree

59.0.0 2022-10-10

  • ⚠️ BREAKING: removed CommandLine app 'jtree'

58.0.0 2022-10-10

  • ⚠️ BREAKING: removed AbstractBuilder
  • 🏥 more path fixes for Windows users

57.0.0 2022-10-10

  • ⚠️ BREAKING: removed rarely commands from CLI app: 'usage', 'programs', 'allHistory', 'distribute', 'combined'
